Jumble Dumble - Word Unscramble Game for Android using Jetpack Compose UI toolkit

Jumble Dumble is an engaging Android application designed to challenge and enhance your word-solving skills. Dive into the world of jumbled words and test your vocabulary as you unscramble ten words in each round. This project aims to provide an entertaining and educational experience for users of all ages.

Key Features:

1. Word Unscramble Challenge: Users are presented with ten jumbled words in each round. The goal is to decipher the correct word from the given mix of letters.
2. Randomized Word Generation: The application dynamically generates a diverse set of words for each round, ensuring a fresh and challenging experience with every play.
3. Scoring System: Keep track of your progress with a built-in scoring system. Earn 10 points for each correct answer and strive to improve your score with each attempt.
4. User-Friendly Interface: Jumble Dumble features an intuitive and visually appealing interface, making it easy for users to navigate through the game effortlessly.
